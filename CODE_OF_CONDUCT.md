# Code of Conduct

## Our pledge

We are committed to making participation in this project a welcoming experience for everyone, regardless of background or experience level.

## Our standards

**Constructive behavior includes:**
- Reporting bugs with clear reproduction steps
- Suggesting improvements with specific failure-mode framing
- Giving and receiving technical feedback directly and respectfully
- Acknowledging when something works well

**Unacceptable behavior includes:**
- Harassment, personal attacks, or dismissive comments
- Deliberately publishing others' private information
- Conduct that would be inappropriate in a professional setting

## Scope

This code of conduct applies to all project spaces: GitHub issues, pull requests, and discussions.

## Enforcement

Report issues to the maintainers via the [security disclosure process](SECURITY.md#responsible-disclosure) or by emailing the address in the package.json. Reports will be reviewed and responded to.

This project uses the [Contributor Covenant](https://www.contributor-covenant.org/) as its base, adapted for a small technical project.
